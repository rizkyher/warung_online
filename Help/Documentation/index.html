Product Description

You have 4 short codes.
You could use Editor mode or Show HTML mode

Product New Badge short code. Put day limit, how many days show new badge in your product.

Product description short code

Product description short code in Product description tab

Affiliate link product short code in "Add to cart icon"


  1. [new_products]Here add your product New Badge day limit[/new_products] - work on Product Grid and List view
  2. [short_description]Here add your product short description[/short_description] - work on Product page below the product condition area , List product list view page, Search page and quick view
  3. [product_description]Here add your product full description[/product_description] - work on Product page in product description tab
  4. [affiliate_link]Here add your affiliate link[/affiliate_link] - work on Product page in "Add to cart icon"
